FEATURES
================================================================================

1. AUTO-HIGHLIGHT
   ✓ Sidebar link highlights when you scroll to that section
   ✓ Blue color and bold text for active section
   ✓ Left border indicator

2. AUTO-SCROLL SIDEBAR
   ✓ Sidebar scrolls to keep active link visible
   ✓ Smooth animation
   ✓ No manual scrolling needed

3. SMOOTH NAVIGATION
   ✓ Click sidebar link → smooth scroll to section
   ✓ No page jumps
   ✓ Professional user experience

4. RESPONSIVE
   ✓ Works on desktop
   ✓ Hides on mobile (hamburger menu)
   ✓ Touch-friendly

================================================================================
CODE STRUCTURE
================================================================================

HTML Structure:
<div class="sidebar">
    <a href="#weather">Weather</a>          ← Links to sections
    <a href="#crop">Crop Recommendation</a>
</div>

<div class="main-content">
    <section id="weather">...</section>     ← Sections with IDs
    <section id="crop">...</section>
</div>

JavaScript Logic:
1. Detect scroll position
2. Find which section is in view
3. Add 'active' class to matching sidebar link
4. Remove 'active' from other links
5. Scroll sidebar to show active link

CSS Styling:
.sidebar a.active {
    color: #3498db;           /* Blue text */
    font-weight: 600;         /* Bold */
    border-left: 3px solid;   /* Blue indicator */
}
