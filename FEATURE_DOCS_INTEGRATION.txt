STEP 1: Add Feature Explanations to Dashboard
================================================================================

1. In app.py, import the feature explanations:
   
   from feature_explanations import FEATURE_EXPLANATIONS

2. In your dashboard() function, add to dashboard_data:
   
   dashboard_data = {
       'user': user,
       'weather': weather,
       # ... existing data ...
       'feature_explanations': FEATURE_EXPLANATIONS  # Add this line
   }

3. In your dashboard.html, add info icons to section headers:
   
   <h2>
       <i class="fas fa-cloud"></i> Weather Forecast
       <i class="fas fa-info-circle info-icon" onclick="toggleInfo('weather-info')"></i>
   </h2>

4. Add collapsible explanation below each header:
   
   <div id="weather-info" class="feature-explanation collapsible-info">
       <h4><i class="fas fa-lightbulb"></i> What is this?</h4>
       <p>{{ feature_explanations.weather.description }}</p>
       <div class="how-it-works">
           <strong>How it works:</strong> {{ feature_explanations.weather.how_it_works }}
       </div>
   </div>

5. Copy the CSS and JavaScript from feature_explanations_snippet.html

================================================================================
STEP 3: Create Documentation Route
================================================================================

Add this route to app.py:

@app.route('/docs')
def documentation():
    """Display project documentation"""
    with open('PROJECT_DOCUMENTATION.txt', 'r') as f:
        doc_content = f.read()
    return render_template('documentation.html', content=doc_content)
