FEATURE 1: SMS/WhatsApp Alerts System
================================================================================

DESCRIPTION:
- Send automated alerts to farmers via SMS or WhatsApp
- Alerts based on real-time weather, soil moisture, market prices
- No internet needed on farmer's phone

DATABASE MODEL: Alert
- Stores all alerts sent to users
- Tracks delivery status (pending/sent/failed)
- Alert types: weather, irrigation, price, pest

API ENDPOINTS:
1. POST /api/alerts/send
   - Send custom alert to farmer
   - Body: {type, message, method: 'sms'/'whatsapp'}

2. GET /api/alerts/history
   - Get all alerts sent to user
   - Returns: alert history with timestamps

3. POST /api/alerts/auto-generate
   - Auto-create alerts based on conditions
   - Checks: temperature, soil moisture, rain forecast
   - Automatically sends if conditions met

DYNAMIC FEATURES:
- Auto-generates alerts when:
  * Temperature > 35°C (heat alert)
  * Soil moisture < 40% (irrigation alert)
  * Heavy rain forecast > 20mm (rain alert)
  * Market price changes > 10% (price alert)

FEATURE 2: Community Marketplace
================================================================================

DESCRIPTION:
- Farmers can buy/sell crops directly (no middlemen)
- Rent/share equipment with nearby farmers
- Hire labor from local community
- Group buying for seeds/fertilizers

DATABASE MODEL: MarketplaceListing
- listing_type: sell, buy, rent, hire
- category: crop, equipment, labor, seeds
- Includes price, quantity, location, contact
- Status tracking: active, sold, closed

API ENDPOINTS:
1. GET /api/marketplace/listings
   - Get all active listings
   - Filter by: category, type, location
   - Returns: list of available items

2. POST /api/marketplace/create
   - Create new listing
   - Body: {type, category, title, description, price, quantity}

3. GET /api/marketplace/my-listings
   - Get user's own listings
   - Track what you're selling/buying

4. POST /api/marketplace/close/<listing_id>
   - Mark listing as sold/closed

DYNAMIC FEATURES:
- Real-time marketplace updates
- Location-based listings (nearby farmers)
- Price comparison with market rates
- Contact seller directly via phone

EXAMPLE LISTINGS:
- "Fresh Wheat - 100 Quintal @ Rs 2400/quintal"
- "Tractor for Rent - Rs 1200/day"
- "Need 5 Workers for Harvesting - Rs 500/day"

FEATURE 3: Crop Disease Detection via Camera
================================================================================

DESCRIPTION:
- Take photo of diseased leaf/plant
- AI detects disease instantly
- Get treatment recommendations
- Track disease history

DATABASE MODEL: DiseaseDetection
- Stores: crop, disease name, confidence, severity
- Treatment recommendations
- Image storage
- Resolution tracking

API ENDPOINTS:
1. POST /api/disease/detect
   - Upload image of crop/leaf
   - Body: multipart/form-data with image file
   - Returns: disease name, confidence %, treatment

2. GET /api/disease/history
   - Get all past detections
   - Track recurring diseases

3. POST /api/disease/resolve/<detection_id>
   - Mark disease as treated/resolved

DYNAMIC FEATURES:
- AI-powered disease detection
- Confidence score (75-98%)
- Severity levels: low, medium, high
- Specific treatment for each disease
- Image history for comparison

DISEASES DETECTED:
- Leaf Blight
- Aphid Infestation
- Powdery Mildew
- Rust
- Bacterial Wilt
- And more...

TREATMENT EXAMPLES:
- "Apply copper-based fungicide. Remove infected leaves."
- "Use neem oil spray. Introduce ladybugs."
- "Improve air circulation. Apply sulfur-based fungicide."

FEATURE 4: Loan & Insurance Integration
================================================================================

DESCRIPTION:
- Check loan eligibility instantly
- Apply for Kisan Credit Card
- Compare crop insurance plans
- Track application status

DATABASE MODELS:
1. LoanApplication
   - loan_type: kisan_credit, crop_loan, equipment
   - Amount, purpose, status
   - Interest rate, tenure

2. Insurance
   - scheme_name, crop_covered
   - Sum insured, premium
   - Start/end dates, status

API ENDPOINTS:

LOANS:
1. GET /api/loan/eligibility
   - Check if eligible for loan
   - Returns: max amount, interest rate, reason

2. POST /api/loan/apply
   - Apply for loan
   - Body: {loan_type, amount, purpose, tenure}

3. GET /api/loan/history
   - Track all loan applications
   - Status: pending, approved, rejected

INSURANCE:
1. GET /api/insurance/plans
   - Get available insurance schemes
   - Compare premium rates

2. POST /api/insurance/apply
   - Apply for crop insurance
   - Body: {scheme_name, crop, sum_insured, premium}

3. GET /api/insurance/my-policies
   - View active insurance policies

DYNAMIC FEATURES:
- Eligibility calculated based on:
  * Farm size
  * Income history
  * Expense records
- Max loan = Farm size × Rs 50,000
- Interest rate: 7.5% (subsidized)

INSURANCE PLANS:
1. PM Fasal Bima Yojana - 2% premium
2. Weather Based Crop Insurance - 3.5% premium
3. Comprehensive Crop Insurance - 5% premium

FEATURE 5: Video Learning Library
================================================================================

DESCRIPTION:
- Short farming technique videos (2-5 min)
- Success stories from other farmers
- Government scheme tutorials
- Multilingual support (8 languages)
- Offline download option
- Track watch progress

DATABASE MODELS:
1. VideoLibrary
   - title, description, category
   - language, duration
   - video_url, thumbnail
   - views, likes

2. VideoProgress
   - Tracks user's watch history
   - Progress percentage
   - Completion status

API ENDPOINTS:
1. GET /api/videos/library
   - Get all videos
   - Filter by: category, language
   - Sort by: views, likes, date

2. POST /api/videos/watch/<video_id>
   - Start watching video
   - Increments view count
   - Returns: video URL, current progress

3. POST /api/videos/progress/<video_id>
   - Update watch progress
   - Body: {progress: 0-100}
   - Marks completed if progress >= 90%

4. GET /api/videos/my-progress
   - Get watch history
   - See completed videos

5. POST /api/videos/like/<video_id>
   - Like a video

DYNAMIC FEATURES:
- Personalized recommendations
- Continue watching from where you left
- Track learning progress
- Popular videos (most viewed)
- Trending videos (most liked)

VIDEO CATEGORIES:
- Farming Techniques
- Success Stories
- Government Schemes
- Organic Farming
- Equipment Usage
- Pest Management
- Soil Health

LANGUAGES SUPPORTED:
