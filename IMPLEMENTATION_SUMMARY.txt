HOW TO INTEGRATE (3 SIMPLE STEPS)
================================================================================

STEP 1: Add Models to app.py
-----------------------------
1. Open app.py
2. Find: class Notification(db.Model):
3. After it, copy-paste ALL models from new_features.py:
   - Alert
   - MarketplaceListing
   - DiseaseDetection
   - LoanApplication
   - Insurance
   - VideoLibrary
   - VideoProgress

STEP 2: Add Routes to app.py
-----------------------------
1. In app.py, find: if __name__ == '__main__':
2. BEFORE that line, copy-paste ALL routes from new_routes.py
3. Also add helper functions from new_features.py:
   - send_sms_alert()
   - send_whatsapp_alert()
   - detect_disease_from_image()
   - calculate_loan_eligibility()
   - get_insurance_plans()

STEP 3: Initialize Database & Add Sample Data
----------------------------------------------
1. Stop Flask app if running
2. Run: python add_sample_data.py
3. Start Flask app: python app.py
4. Test APIs using browser/Postman
